Instalacja
^^^^^^^^^^^^

Możesz zainstalować ``vulcan-api`` za pomocą ``pip``

.. code:: console

    $ pip install vulcan-api

lub ręcznie przy użyciu ``flit``

.. code:: console

    $ pip install flit
    $ git clone https://github.com/kapi2289/vulcan-api.git
    $ cd vulcan-api
    $ python -m flit install
